Policy Links — Arc Diagram Prototype

A layout alternative to the force graph. Policies are ordered top→bottom by merged category (Economic Security → Global); vertical position encodes only category, so nothing is left to accidental physics. Arcs are policy links (pairsWith). Hover or focus a node to trace its links; click for detail. This is a visual prototype — the real edge set gets overhauled later.
